# Activities and Hands-On Exercises

This document provides a set of practical exercises to reinforce concepts in data handling, visualization, and basic statistics.

## Exercises Overview

### 1. Load a CSV Dataset

* Example datasets: student grades, weather data, or any dataset of interest.
* Use Python (pandas) or spreadsheet software (Google Sheets, Excel) to load the data.
* Explore the dataset: check column names, data types, and summary statistics.

### 2. Create Histograms and Scatter Plots

* Use Matplotlib or Seaborn in Python, or chart tools in Sheets.
* Histograms: Visualize the distribution of a numeric variable.
* Scatter plots: Explore relationships between two numeric variables.
* Label axes and add titles for clarity.

### 3. Compute Basic Statistics

* Write simple scripts or formulas to calculate:

  * Mean (average)
  * Range (max - min)
* Apply these calculations to numeric columns in your dataset.

### 4. Mini-Project: Collect and Visualize Class Data

* Collect data from classmates (e.g., favorite numbers, commute times, or study hours).
* Organize the data in a table or CSV file.
* Visualize using:

  * Histograms for single-variable distributions
  * Scatter plots for relationships between variables
* Draw insights from your visualizations and summarize findings.

## Note on Code

All Python code to perform these exercises can be written in a separate script file, e.g., `activities_hands_on.py`. This README focuses on the tasks and conceptual workflow.
